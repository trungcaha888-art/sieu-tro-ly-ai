# Skill: Tao Banner Chuyen Nghiep

## Khi nao dung

- User yeu cau tao banner, poster, hero image, thumbnail
- User nhac den "giong poster gioi thieu chuyen nghiep"
- User muon co mat anh Trung trong banner

## Input can xac dinh

1. Banner dung o dau:
   - hero tren web
   - og image
   - poster quang cao
2. Anh tham chieu nao se duoc dung:
   - `anh 4.png`
   - `anh 5.png`
   - `anh 6.png`
3. Thong diep chinh:
   - "Sieu Tro Ly AI"
   - "Tu dong hoa 80% cong viec"
   - "Webinar mien phi"

## Cach xu ly

### Truong hop co cong cu tao anh

- Tao prompt rat cu the, nhan manh:
  - poster chuyen nghiep
  - bo cuc hero ngang
  - nhan vat chinh la anh Trung tu anh tham chieu
  - tong mau cong nghe, cao cap, sach
  - text de trong vung an toan de sau nay chen tren web

### Truong hop khong co cong cu tao anh

- Van phai tra ra:
  - ten file can tao, vi du `banner-hero.png`
  - prompt chi tiet de copy-paste qua cong cu tao anh
  - huong dan file nao trong HTML/CSS can cap nhat de dung anh do

## Mau prompt uu tien

```text
Tao mot banner hero chuyen nghiep cho landing page webinar "Sieu Tro Ly AI".
Phong cach cao cap, hien dai, cong nghe, giong poster gioi thieu dich vu AI.
Nhan vat chinh la mot doanh nhan nam nguoi Viet, lay giong guong mat va than thai tu anh tham chieu.
Bo cuc ngang 16:9, nhan vat dung lech phai, ben trai de khoang trong de dat tieu de.
Background la gradient toi xanh navy + teal, co glow cong nghe, dashboard AI mo ao, icon chat/Zalo/Telegram nhe o hau canh.
Cam giac tin cay, thong minh, premium, khong bi loe loet.
Khong dat qua nhieu chu vao anh, chi can 1 dong nho neu can: "Sieu Tro Ly AI".
Anh net, advertising quality, realistic, studio lighting, high-end keynote poster.
```

## Output mong muon

- Prompt hoan chinh
- Ten file de luu vao workspace
- Neu can, de xuat cap nhat `index.html` / `style.css`
