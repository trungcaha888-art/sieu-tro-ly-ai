# Workspace Rules - AI Tro Ly Landing Page

## Muc tieu

Workspace nay dung de:

- chinh sua landing page `index.html` va `style.css`
- tao hoac cap nhat asset banner/poster cho trang
- cap nhat `og-image.png` va cac anh minh hoa lien quan

## Quy tac lam viec

1. Neu user nhac den `banner`, `poster`, `anh`, `hero`, `og-image` thi uu tien xu ly nhu mot bai toan tao/chinh sua hinh anh cho landing page nay.
2. Neu can tao anh moi, phai dua ra prompt ro rang, bo cuc ro rang, va uu tien dung anh tham chieu san co trong folder hien tai.
3. Neu user muon thay doi giao dien, luon cap nhat file HTML/CSS trong workspace nay, khong dua huong dan chung chung.
4. Asset moi phai luu ngay trong folder:
   - `og-image.png` neu la anh share link
   - hoac ten ro rang nhu `banner-hero.png`, `poster-webinar.png`
5. Khi dung anh tham chieu nguoi that, uu tien cac file san co:
   - `anh 4.png`
   - `anh 5.png`
   - `anh 6.png`
6. Neu khong co cong cu tao anh truc tiep, van phai:
   - viet prompt san xuat anh chi tiet
   - chi ro ten file can tao
   - cap nhat HTML/CSS de dung dung ten file do

## Files quan trong

- `index.html`: landing page chinh
- `style.css`: giao dien
- `og-image.png`: thumbnail chia se
- `content.txt`: noi dung goc
